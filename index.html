Blog Curso Git

  • clear: limpiar terminal (control+l)
  • ls: ver archivos y carpetas
  • git init: inicializa el repositorio
  • status: estado del repositorio
  • git add: mueve a staging area
  • commit: crea la nueva version
  • git log: se muestran los commits
  • checkout volver a version anterior
  • reset/(--hard): deshacer los cambios locales en el estado de un repositorio / borra las modificaciones en vscode
  • rm/(-r): remove / remueve todo
  • restore: revierte el rm (NO EL RESET)
  • merge: fusiona ramas
